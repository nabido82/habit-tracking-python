# Habit Tracking Application

This project is a simple habit tracking application developed in Python as part of a bachelor-level course.

The application allows users to create habits, mark them as completed, and analyze their progress over time.

## Features
- Add daily or weekly habits
- Mark habits as completed
- Store habit data
- View habit list and progress
- Basic analytics
- Unit tests using pytest

## Project Structure
- main.py: Entry point of the application
- habit.py: Defines the Habit class
- habit_tracker.py: Manages habits
- storage.py: Handles data storage
- analytics.py: Provides habit statistics
- test_habits.py: Contains unit tests

## How to Run
Run the application by executing:

python main.py

## Notes
This project focuses on object-oriented design and clean code structure.
